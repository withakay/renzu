# Core

## Purpose
Foundation infrastructure for the Code Context Service. Provides configuration management, structured logging, health endpoints, shared types, and dependency injection.

## Scope
- Configuration (env vars, settings classes)
- Structured JSON logging
- Health check endpoints (`/healthz`, `/readyz`)
- Shared type definitions and protocols
- Dependency injection container
- Error handling base classes
