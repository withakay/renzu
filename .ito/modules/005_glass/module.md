# Glass

## Purpose
Integration with Glean and Glass servers for precise code navigation. Provides symbol-level operations like listing document symbols, finding references, and describing definitions.

## Scope
- Glean/Glass server connection management
- Glass client (Thrift or HTTP proxy)
- Document symbols listing
- Symbol definition lookup
- Find references for symbols
