# SCIP

## Purpose
SCIP index parsing and symbol-aware chunking. Enhances tree-sitter chunking with precise symbol boundaries.

## Scope
- SCIP index file parsing (protobuf)
- Symbol definition span extraction
- Hybrid chunking (tree-sitter + SCIP)
- Chunk merging and deduplication
